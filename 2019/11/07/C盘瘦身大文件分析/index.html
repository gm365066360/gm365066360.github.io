C盘瘦身大文件分析

Posted on 2019-11-07

系统盘C盘瘦身 节省50G空间 大文件分析

####磁盘大文件分析工具 WinDirStat

  • #####根据色块大小, 我们会发现两个庞然大物
  • pagefile.sys 9G系统虚拟内存分页文件,不可删除,可转移D盘,需重启

    参考文章

  • hiberfil.sys 6G系统休眠文件, 可命令行关闭休眠即可删除文件

    系统盘文件夹Windows/system32中的cmd.exe文件 右击在弹出菜单中，点击“以管理员身份运行”。然后执行命令行powercfg -h off，即可成功删除Windows系统休眠文件hiberfil.sys

  • #####关于休眠与睡眠 microsoft
    对于许多电脑（尤其是笔记本电脑和平板电脑），当你合上显示屏或按下电源按钮后，电脑就会进入睡眠状态。

####关于AndroidStudio在系统盘相关文件 C:\Users..\

  • #####.android\avd 9G
    下载过的虚拟机文件,贼大,一个就9G, 直接干掉, 用真机测试
  • #####.gradle 9G
    转移大法, 并新建系统变量GRADLE_USER_HOME 确保使用前斜杠/

  • #####.AndroidStudio3.5 config&system文件夹 2G

    转移大法, 参考文章
    修改AndroidStudio安装路径\bin\idea.properties文件：
    idea.config.path=D:/android/ide/cache/.AndroidStudio3.0/config
    idea.system.path=D:/android/ide/cache/.AndroidStudio3.0/system
    #####注意!注意!注意! 去掉前面的注释符#号 确保使用前斜杠/
